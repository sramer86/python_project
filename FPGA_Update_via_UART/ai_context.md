请用 Python 编写一个 FPGA 串口在线升级上位机工具，要求如下：

1. **使用 tkinter 做 GUI 界面**，界面简洁清晰，符合工业工具风格。
2. 功能模块：
   - 串口配置区：串口号选择、波特率设置（可输入或下拉常见 9600/115200/230400/460800/921600）、打开/关闭串口按钮。

   - 操作区：【开始升级】按钮、【回读 Flash】按钮。
   - 进度与日志区：进度条实时显示烧录进度、文本框显示日志信息。
3. **烧录流程**
   - 点击【开始升级】后，读取选中的 .bin 文件。
   - 将.bin文件的大小除以4KB结果向上取整，例如文件大小1.351MB，除以4KB后向上取整就等于338，作为擦除扇区的计数。

   - 准备烧写程序，每次读取.bin文件256个字节进行编程，每次编程前发送“写使能”命令，再发送“页编程”命令，以烧写的起始地址为编程的起始地址，每次递增256B地址，直致.bit文件结束，每次发送完一个命令都要等待串口回传，如果接到的数据有连续5个0x66，则表示操作成功，如果接到的数据有连续5个0xEE，则表示发送错误，需要重新发送上一条命令，如果在100ms时间内没有收到任何有用数据，则退出当前流程，并报告超时断开警告，数据实时跟新编程进度条。
   - 按照自定义串口升级协议发送数据，协议帧结构预见第5条。
   - 实时更新进度条（百分比），擦除和编程均显示一个进度条，只是名字不同。
   - 烧录完成后弹出提示框：“升级完成”。
   - 异常处理：超时、校验错误、串口断开等给出明确提示。
4. **回读 Flash 功能**
   - 输入回读起始地址、回读长度。
   - 从 FPGA 回读 Flash 数据。
   - 将回读数据保存为 **.bit 文件**。
   - 计算并显示回读数据的 **校验和（计算CRC32）**。
5. **协议接口预留**
